النظام بيستخدم Microsoft Graph API لإرسال الإيميلات من حسابك مباشرة.
لازم تسجّل تطبيق في Azure Portal وتحط الـ Client ID هنا:

⚙️ خطوات إعداد Azure بسيطة:
1. افتح portal.azure.com
2. Azure Active Directory → App Registrations → New Registration
3. اسم التطبيق: Dashboard Mailer — Redirect URI: ${window.location.origin}
4. API Permissions → Add → Microsoft Graph → Delegated → Mail.Send
5. انسخ الـ Application (client) ID وحطه فوق
